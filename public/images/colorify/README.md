# Imagens COLORIFY Landing Page

Coloque aqui as imagens de demonstração para a landing page:

## Arquivos necessários:

1. **`hero-demo.gif`** - GIF mostrando o app funcionando na prática (será exibido na seção Hero)

2. **Exemplos de transformações** (foto colorida → desenho preto e branco):
   - `example-1.jpg` - Exemplo de transformação (crianças)
   - `example-2.jpg` - Exemplo de transformação (pet/cachorro)
   - `example-3.jpg` - Exemplo de transformação (família)
   - `example-4.jpg` - Exemplo de transformação (momentos especiais)

## Onde as imagens aparecem:

- **hero-demo.gif**: Seção Hero, logo após a descrição do produto
- **example-*.jpg**: Seção "O que pode ser transformado", mostrando exemplos visuais das transformações

## Formato recomendado:

- GIF: Máximo 5MB para melhor performance
- Imagens: JPG ou PNG, recomendado 1200x800px ou similar
- As imagens serão exibidas automaticamente quando adicionadas nesta pasta

A landing page está em `/colorify-landing` e as imagens serão carregadas automaticamente quando você adicionar os arquivos aqui.
